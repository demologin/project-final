# Jira Rush App

## Запуск

Запустить файл [docker-compose.yml](docker-compose.yml)

- Или выполните команду

```bash
docker compose up
```  

Для изменения режима DEV PROD измените строку 12 в [Dockerfile](Dockerfile)
